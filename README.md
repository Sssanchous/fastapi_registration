## FASTAPI регистрация

1. Установка виртуального окружения
```
1) python -m venv venv
```
```
2) venv\Scripts\activate
```
3. Установка зависимостей
```
pip install -r requirements.txt
```
3.  Запуск сервера
```
uvicorn main:app --reload
```
